# visidata-uproot
visidata loader for root files, based on uproot

## Features

- [x] ROOT files
  - [x] subdirectoreis
  - [x] count of objects
- [ ] TH
  - [x] TH1
    - [x] edges, heitghts
    - [x] errors, variance, counts
  - [ ] TH2
  - [ ] TH3
- [ ] TTree
  - [x] simple ntuples

## Usage

- Copy `root.py` to `~/.visidata/`
- Edit `~/.visidatarc`

```python
import root # loads ~/.visidata/root.py
```
